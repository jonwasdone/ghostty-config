# Ghostty Config

My personal configuration for [Ghostty](https://ghostty.org/), a fast, native, feature-rich terminal emulator.

## Features

- **Font**: Inconsolata LGC Nerd Font Complete Mono (size 18)
- **Theme**: Catppuccin Macchiato
- **Background**: Custom background images with opacity
- **Shell Integration**: Zsh with sudo support
- **macOS Optimizations**: Transparent titlebar, tabs, Display P3 colorspace

## What's Included

- `config` - Main Ghostty configuration file
- `backgrounds/` - Background images (Bloodborne, Elden Ring)
- `themes/` - Catppuccin Macchiato theme

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/jonwasdone/ghostty-config.git ~/.config/ghostty
   ```

2. Make sure you have the required font installed:
   - [Inconsolata LGC Nerd Font](https://www.nerdfonts.com/)

3. Restart Ghostty or reload the configuration

## Key Features

- **Custom Keybindings**:
  - `Ctrl+W` (global): Toggle quick terminal
  - `Shift+Enter`: Send escape sequence

- **Visual Enhancements**:
  - Background image with 20% opacity
  - Display P3 wide color gamut support
  - Custom cursor color
  - Transparent macOS titlebar with tabs

- **Enhanced Usability**:
  - Hide mouse while typing
  - Increased scroll multiplier (2x)
  - Balanced window padding
  - Window state persistence
